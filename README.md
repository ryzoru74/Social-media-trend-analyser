
# Social Media Trend Analyzer CLI

Stop guessing what to post. Get data-driven content ideas for TikTok, YouTube, and Instagram in seconds.

The **Social Media Trend Analyzer** is a powerful, interactive CLI tool designed for content creators who need to stay ahead of the curve. Stop manually researching trends and start focusing on creating content that your audience actually wants to watch.

---

## 🚀 Features

*   **Multi-Platform Support:** Get trend data for **TikTok**, **YouTube**, and **Instagram**.
*   **Interactive CLI:** No complicated flags. Just answer simple prompts to get your results.
*   **Data-Driven Insights:** Get 10 fresh, relevant content ideas for any niche you enter.
*   **Clean Output:** Beautiful, readable tables right in your terminal.
*   **Lightning Fast:** Save hours of manual research time every single week.

---

## 🛠️ Quick Start

### Prerequisites
*   Python 3.8+ installed on your system.

### Installation

1. Download the source code.
2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

### Usage

Run the tool from your terminal:

```bash
python3 src/cli.py
```

Follow the interactive prompts:
1. Select your platform (1. Youtube, 2. Instagram, 3. Tiktok).
2. Enter your niche (e.g., `gaming`, `cooking`, `tech`).
